## Project Overview
This project presents an Excel-based Retail Sales Analysis using pivot tables and an interactive dashboard.  
The goal of this project is to analyze retail sales data and extract meaningful business insights using Excel.

---

## Tools Used
- Microsoft Excel / Excel Online
- Pivot Tables
- Excel Charts
---

## Dataset Description
The dataset contains retail sales transactions with the following columns:
- Order_Date
- Product
- Category
- Quantity
- Sales
- Region

---

## Analysis Performed
The following analyses were performed using pivot tables:
- Total Sales summary
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Top 5 Products by Sales

---

## Dashboard Features
The Excel dashboard includes:
- KPI cards for Total Sales and Total Quantity
- Sales by Category (Column Chart)
- Sales by Region (Bar Chart)
- Monthly Sales Trend (Line Chart)
- Top 5 Products by Sales(Column Chart)

---

## Project Structure
- Excel file containing raw data, pivot tables, and dashboard
- Markdown file documenting dashboard and pivot table screenshots
- Project explanation file describing the implementation in detail

---

## Project Outcome
This project demonstrates practical Excel skills including data preparation, pivot table analysis, and dashboard creation.  
It highlights the ability to transform raw data into meaningful insights using Excel.
